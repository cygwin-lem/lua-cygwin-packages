An experimental Cygwin tree for Lua

https://cygwin-lem.github.io/lua-cygwin-packages/lua-cygwin/

  1. Start setup-*.exe with an option -X.
  2. Select Install from Internet.
  3. Input User URL https://cygwin-lem.github.io/lua-cygwin-packages/lua-cygwin/ and push Add button.
  4. Push Next button.
  5. Select packages you want...
  6. ...

Note

  • lua: Turned into a meta-package for Lua default version
    • current is for lua53
    • test is for lua54
  • lua54, lua53, lua52, lua51: Multiple versions of Lua can be installed.
  • luarocks: Installs rocks and modules to an alternative tree structure
    • luarocks tree: /usr
    • rocks: /usr/lib/lua/luarocks/rocks-${LUA_VERSION}
    • manifest: /usr/lib/lua/luarocks/rocks-${LUA_VERSION}/manifest
    • deploy_bin_dir: /usr/lib/lua/luarocks/bin/${LUA_VERSION}
    • deploy_lib_dir: /usr/lib/lua/luarocks/${LUA_VERSION}
    • deploy_lua_dir: /usr/share/lua/luarocks/${LUA_VERSION}
  • lua-luadoc: obsoleted by lua-ldoc, not updated
  • cantor-backend-lua: not updated, part of cantor
  • hexchat-lua: not updated, part of hexchat
  • httpd_mod_lua: not updated, part of httpd
  • lighttpd-lua-modules: not updated, part of lighttpd
  • lua-gv: not updated, part of graphviz
  • lua-xapian: not updated, part of xapian-bindings
  • lua5.1-keybinder: not updated, part of keybinder
  • weechat-lua: not updated, part of weechat

Usage Examples of LuaRocks

Here are usage examples of luarocks, or try a test script test-luarocks.sh (view in html). 
luarocks-5.4 install argparse \
&& luarocks-5.4 list \
&& lua5.4 -e 'require("argparse")' && echo OK: argparse \
&& luarocks-5.4 remove argparse

luarocks-5.4 install lua-cjson \
&& luarocks-5.4 list \
&& lua5.4 -e 'require("cjson")' && echo OK: cjson\
&& lua5.4 -e 'require("cjson.util")' && echo OK: cjson.util \
&& luarocks-5.4 remove lua-cjson

luarocks-5.3 install argparse \
&& luarocks-5.3 list \
&& lua5.3 -e 'require("argparse")' && echo OK: argparse \
&& luarocks-5.3 remove argparse

luarocks-5.3 install lua-cjson \
&& luarocks-5.3 list \
&& lua5.3 -e 'require("cjson")' && echo OK: cjson \
&& lua5.3 -e 'require("cjson.util")' && echo OK: cjson.util \
&& luarocks-5.3 remove lua-cjson
